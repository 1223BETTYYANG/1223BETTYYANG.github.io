## How to Deploy on GitHub Pages

1.  **Create a GitHub Repository**:
    *   Go to [GitHub](https://github.com) and create a new repository.
    *   Name it `your-username.github.io` (replace `your-username` with your actual GitHub username). This is important for the default GitHub Pages URL structure.

2.  **Upload Files**:
    *   Upload all the files from this folder (`index.html`, `assets/`, etc.) to your new repository.
    *   You can do this via the command line (git push) or by dragging and dropping files on the GitHub website.

3.  **Enable GitHub Pages** (if not using the `username.github.io` convention):
    *   Go to your repository **Settings**.
    *   Click on **Pages** in the left sidebar.
    *   Under **Source**, select `main` (or `master`) branch.
    *   Click **Save**.

4.  **View Your Site**:
    *   After a few minutes, your site will be live at `https://your-username.github.io`.

## Customization

*   **Profile Image**: Replace `assets/images/profile-placeholder.png` with your actual photo. Make sure to update the `src` in `index.html` if the filename is different.
*   **Links**: Update the `href` attributes in `index.html` for your email, LinkedIn, GitHub, etc.
*   **Content**: Edit `index.html` to update your information as you gain new experience.
